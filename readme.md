Guide d'installation

Le code source est sur ce lien github : https://github.com/FabianCoder047/GPB

Installez wamp 
Installez git sur votre machine
Allez dans le dossier wamp64 et ensuite dans www
Faites clic droit et ouvrir dans le Terminal

Cloner le projet en tapant la commande ci-dessous : 
git clone https://github.com/FabianCoder047/GPB

Après le clonage vous aurez un dossier GPB dans le dossier www : le code du projet est à l'intérieur

Démarrez ensuite wamp
Allez au phpmyadmin via cette url: http://localhost/phpmyadmin/index.php

Créer une base de données gpb

Importez la base de données gpb.sql située dans le dossier GPB.

Lancer le logiciel via l'url suivante : http://localhost/gpb/index.php
